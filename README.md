# React-learn

Learn React and work toward React mastery with this simple path:

1. **Fundamentals**  
   Learn JSX, components, props, state, events, and conditional rendering.
2. **Core React Skills**  
   Practice hooks (`useState`, `useEffect`, `useMemo`, `useCallback`) and component composition.
3. **Routing & State Management**  
   Build multi-page apps with React Router and manage shared state with Context or Redux Toolkit.
4. **API Integration**  
   Fetch data, handle loading/error states, and build reusable data-fetching patterns.
5. **Performance & Best Practices**  
   Optimize re-renders, split code, and organize scalable folder structures.
6. **Project Practice**  
   Build real projects (dashboard, e-commerce UI, blog app) and deploy them.

Consistency is the key: build, review, and improve projects every week.
